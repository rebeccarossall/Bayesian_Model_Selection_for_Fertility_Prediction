# Bayesian Statistical Modelling for Fertility Prediction

This project investigates the relationships between fertility rates and demographic variables across 47 Swiss regions using Bayesian statistical modelling in R. The analysis combines exploratory data analysis (EDA), Bayesian inference, Bayesian model selection, and predictive modelling to identify the strongest predictors of fertility and evaluate model performance.

This project is based on a final-year university Bayesian statistics assignment that achieved a mark of **94%**.


## Skills Demonstrated

- Exploratory Data Analysis (EDA)
- Bayesian Statistics
- Bayesian Linear Regression
- Bayesian Inference
- Bayes Factors
- Bayesian Model Selection
- Predictive Modelling
- Feature Engineering
- Model Evaluation (MSE & RMSE)
- Data Visualisation
- R Programming
- Statistical Reporting

## Key Highlights

- Conducted exploratory data analysis (EDA) using scatterplots, histograms, correlation analysis, and variable transformations to investigate relationships between demographic variables and fertility.
- Applied Bayesian linear regression using prior and posterior distributions to estimate model parameters and quantify uncertainty.
- Performed Bayesian model selection by comparing all possible predictor combinations using Bayes Factors.
- Compared competing models using Bayes Factors, predictive accuracy (MSE and RMSE), and model parsimony.
- Generated posterior samples to examine parameter uncertainty and posterior distributions.
- Evaluated predictive performance using an independent test dataset and predicted versus observed comparisons.
