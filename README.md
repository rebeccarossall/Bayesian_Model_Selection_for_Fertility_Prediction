# Bayesian Model Selection for Fertility Prediction

**Awarded 94%** in a University of Nottingham Bayesian statistics project.

This project investigates the relationships between fertility rates and demographic variables across 47 Swiss regions using Bayesian statistical modelling in R. Bayesian linear regression, Bayes Factors and model selection techniques were applied to identify the strongest predictors of fertility and evaluate predictive performance.
